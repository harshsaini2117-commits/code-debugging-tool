# Code Debugging Tool

A Python-based tool designed to identify common syntax and logical errors in code.

## Features
- Detects missing parentheses
- Identifies incorrect use of operators
- Highlights basic logical issues

## Tech Stack
- Python

## How to Run
1. Run the script:
   python debugger.py
2. Enter code as input
3. View detected errors
